Standalone project to monitor and control the xyz stage for the Nikon 
Eclipse 90i microscope. (XY controlled by Corvus)

This project requires:-

from the C Libraries repository:
com_port_control
GCI_utils
string_utils
GL_CVIRegistry
password
status
signals
xml_utils

from the Hardware Control repository:
focus drive

90i.fp and .h from GCI_Libraries

ole32.lib from cvi
