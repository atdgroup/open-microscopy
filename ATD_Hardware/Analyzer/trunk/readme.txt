Standalone project to monitor and control the analyzer for the Nikon 
Eclipse 90i microscope.

This project requires:-

from the C Libraries repository:-
gci_utils.c
gci_utils.h
GL_CVIRegistry.c 
GL_CVIRegistry.h 
signals.c
signals.h
string_utils.h

from the local PC:-
cvi..\sdk\lib\ole32.lib

from GCI_Libraries on public:-
string_utils.lib
90i.fp (to be loaded as a library or instrument)
