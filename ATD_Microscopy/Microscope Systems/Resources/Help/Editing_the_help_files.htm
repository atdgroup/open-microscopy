Editing the help files

Location of the help files
All the help files are located in the following directory:
C:\Devel\ATD\ATD_Microscopy\Microscope Systems\Resources\Help

Editing / adding the help files
In order to edit / add a help file you need to open Microscopy.hhp (using Microsoft HTML Workshop - free software from Microsoft website)

Opening the Contents tab shows the list of items on the help file. Each item can be moved upwards/downwards using the arrows. 



In order to create a new item right click on the heading you want the new item to be created under and click "Insert heading"



Entry title:- name of title to appear in help file (e.g. OpenMicrosocopy).
Add... - Add am HTML file to be associated with this item (when this item will be clicked the HTML file will appear, e.g. OpenMicrosocpy_GUI.html).
Edit... - Edit which HTML is associated with this item (when this item will be clicked the HTML file will appear, e.g. OpenMicrosocpy_GUI.html).
Remove - Removes associated HTML file.

Compiling the help files
Pressing the    button will compile an output help file called Microscopy.chm
*NOTE in order to compile Microscopy.chm has to be closed


The index file (.hhk) can be generated from the page titles etc. using KIT - Keyword Index Tool
