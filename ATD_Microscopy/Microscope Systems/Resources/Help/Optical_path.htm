Optical Path


This is the optical path as defined in Microscope >> Path



The optical path can be chosen from a list
Configure - opens the dialog box (below) where different objective lenses can be added, removed and edited.
Close - closes the dialog box.




Add - adds a new optical path.
Remove - remove an optical path from the list.
Edit - opens the dialog box (below) where specific information regarding the optical path is stored.
Recall - Loads an optical path list from file.
Save - Saves the current optical path list to a file.
Close - closes the dialog box.



Path - name given to the optical path.
OK- closes the dialog box.
