Camera advanced settings (Hamamatsu Cameras)



Black Level

Sets the black level (background level) by changing the brightness setting.

Sensitivity

Not implemented on all cameras.
Sets the EM sensitivity on EM-CCD cameras. CARE - Increased sensitivity shortens the life of the camera. Take notice of warnings displayed in the Log.

Data Mode

Sets the dynamic range of the intensity: 8 bit and 12 bit have 28 and 212 different intensity levels respectively.

Bining

 Binning - pixels in the vertical and horizontal direction are combined. For example 2 x 2 means that that 4 pixels from a 2 x 2 square are combined to a single pixel. The result of binning is that (per pixel) the intensity is increased while the resolution is decreased.

Temp

Not implemented on all cameras.
Displays the sensor temperature on EM-CCD cameras.

CCD Mode

Not implemented on all cameras.
Sets the CCD mode on EM-CCD cameras between normal and the sensitive EM mode (see Sensitivity above).

Photon Imaging

Not implemented on all cameras.
Sets the imaging mode on EM-CCD cameras.

Light Mode

Switches between low and high light mode in terms of gamma. Low-light deactivates the CCD anti-blooming feature and lowers the sensitivity slightly.

Subwindow Size and Location

Preset Sizes -  Sets the size of the subwindow, if the custom setting is chosen the user can set both the size and the location of the subwindow.

Close

Closes the Camera Advanced settings dialog box.
