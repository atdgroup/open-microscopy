Background correction



Should be done at the start of the experiment and if the illumination is not uniform across the field of view.

'Black' Reference - Image should be taken with no illumination present.
'White' Reference - Image should be taken where only the illumination is present (no sample) or with a fluorescent blank if in fluorescence mode.
Preform bg correction - Correction background will be preformed on every image acquired by the camera.
