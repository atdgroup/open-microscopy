Objectives



The objective lens used for imaging can be chosen from a list
Configure - opens the dialog box (below) where different objective lenses can be added, removed and edited.
Close - closes the dialog box.



Add - adds a new objective lens.
Remove - remove an objective lens from the list.
Edit - opens the dialog box (below) where specific information regarding the objective is stored.
Default position - setup the position of the objective lens
Recall - Loads an objective lens list from file.
Save - Saves the current objective lens list to a file
Close - closes the dialog box.




Name - objective name.
Magnification - normally written on the objective.
Medium - air/water/oil...
WD - working distance in mm, normally written on the objective.
NA - Numerical aperture, normally written on the objective.
Back Aperture - Diameter of back aperture in mm.
Condenser - type of condenser.
OK- closes the dialog box.
