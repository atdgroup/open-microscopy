Load and save settings

Reset all modes to default settings - Resets all modes (Microscope >> mode) to the default settings according to the Default*.ini files (e.g. DefaultBrightFieldSettings.ini etc.). These settings are used on startup.

Save settings as...- Saves the current settings for the current mode into a *.ini file.

Load settings from file...- Loads settings for the current mode from a file.

Save settings as default for this mode - Saves the current settings as the default for this mode. These settings will then be used on startup.
