White Laser



Please consult the Fianium manual (available at http://www.fianium.com/supercontinuum.htm).

Back reflection: If this reaches more than 400 the laser will cut itself off automatically; if this happens please check the alarms by pressing the More button.

More: opens the following dialog box.



If laser was automatically turned off the user should check the alarms to determine the cause.

Check Alarms: checks any alarms.
Clear Alarms: clears the alarm log.
Re-Initialise: reinitialise the laser (involves unplugging the laser controller from the computer).
Close: closes the dialog box.
