Scanner

ADVANCED USERS ONLY

This control panel refers to the galvo mirrors which are used to scan the beam in the laser scanning modality (Microscope >> mode >> Laser Scanning).




Start - Start the scan.
Stop - Start the scan.
Acq. frames - Number of frames to acquire, 0 = acquire for ever.
Line / XY scan - Line scan refers to 1D scan, while XY scan refers to a 2D scan.
X shift - Shifts the centre point.
Y shift - Shifts the centre point.
Zoom - Sets the area to be scanned or to park the beam.
Scan speed - Sets the speed of the scan.
Scan resolution - Sets the resolution of the scanned image.
Driver - Must be enabled for the scanner to move.
Scan in progress - Light is on when a scan is in progress.
Error/Standby - Light is on if there is an Error or the scanners are in a Standby mode.
Settings - Pressing the settings button will open the dialog box seen below.

Scanner
XY Axes - Reverse the X and Y axes
Lag - Advanced setting for scanner turn around time.

Acquisition
Pix clk - Switches between an internal and external clock.
Line clk - Switches between an internal and external clock.
Frame clk - Switches between an internal and external clock.

Offset -  (Sometimes used when the galvo's are not aligned properly
x - Amount to offset the  x axis.
y - Amount to offset the  y axis.

OK - Closes the dialog box

Load - Load scanner settings.
Save - Saves scanner settings.
Close - Closes the dialog box.
