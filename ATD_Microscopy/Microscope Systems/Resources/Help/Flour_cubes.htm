Flour cubes


The fluorescent cube is made of three filters: Emission, Dichroic, and Excitation filters. This  fluorescent cube is defined in Microscope >> Cube

The Fluorescent cubes are used when using wide field illumination (Microscope >> Mode >> Fluorescence or Bright Field), when laser scanning is chosen (Microscope >> Mode >> Laser Scanning) the UV2 filter is chosen automatically by the software, and the appropriate filters are chosen using the filter set settings.



Configure - opens the dialog box (below) where different fluorescent cubes can be added, removed and edited.
Close - closes the dialog box.



Add - adds a new fluorescent cube.
Remove - remove a fluorescent cube from the list.
Edit - opens the dialog box (below) where specific information regarding the fluorescent cube is stored.
Recall - Loads a fluorescent cube list from file.
Save - Saves the current fluorescent cube list to a file
Close - closes the dialog box.




Cube name - name of fluorescent cube.
Excitation Min (nm)- Minimum wavelength allowed by the Excitation filter.
Excitation Max (nm) - Maximum wavelength allowed by the Excitation filter.
Dichroic (nm) - Cut-off for the Dichroic filter.
Emission Min (nm) - Minimum wavelength allowed by the Emission filter.
Emission Max (nm) - Maximum wavelength allowed by the Emission filter.
OK- closes the dialog box.
