Pmts



The type of Photo Multiplier Tube (PMT) can be chosen from the list
Configure - opens the dialog box (below) where different PMT can be added, removed and edited.
Close - closes the dialog box.



Add - adds a new PMT
Remove - remove a PMT from the list.
Edit - opens the dialog box (below) where specific information regarding the PMT is stored.
Recall - Loads an PMT list from file.
Save - Saves the current PMT list to a file.
Close - closes the dialog box.



Path - name given to the PMT.
OK- closes the dialog box.
