Stage

The most basic function of the stage panel is to display the current stage position, which is constantly being read.



Stage Moving - Light is on when the stage is moving.

Position
X um - Current position of the stage in the X axis.
Y um - Current position of the stage in the Y axis.

Target
Go to - Go to the position (Move to X um, Move to Y um).
Move to X um - Position to go to in the X axis.
Move to Y um - Position to go to in the X axis.
Move by - Go to by moving with a step size of
X Increment um -  Step size in the X axis
Y Increment um - Step size in the X axis

Datum
The datum is the center point of the stage and we define it as (0,0) in X and Y; it also called a Datum. For convenience, the user may want to re-define the Datum. For example, suppose a group of cells to be scanned lies away from the Datum. Rather than moving the stage around, and noting the new X and Y values at the edges of the ROI, it is often easier to set the center of the ROI as the new Datum. This enables to  begin a scan using symmetric edge co-ordinates e.g. -2000 to 2000 in both X and Y. If a scan has to be repeated over a larger or smaller area it is a simple matter of adjusting these numbers (to say -3000 to 3000) rather than re-calculating the new edge co-ordinates (in terms of the old Datum). The downside of redefining a new Datum is that the absolute co-ordinates have now been lost. Therefore, if the same sample needs to be revisited while the system has been turned off, upon re-initialisation (of the stage) the  Datum will be set as the center of the X and Y axis. One way of overcoming this is to note the X and Y values before the new datum was set, then the process can be repeated.

Set Datum - Sets new Datum position.
GoTo Datum - Go the the Datum position.
Datum diagram - Clicking the mouse anywhere on this diagram will move the stage to that position, green boxes denote the limits of its travel.

Joystick 
When the joystick is enabled the light is on. It is possible to change the speed of the stage. 

Settings... - This will open an extended control panel which is currently password protected (contact Administrator to change these settings)
Re-Initialise - Re-initialisation drives the stage to its limits, set by limit switches (green boxes on Datum diagram). Then it will drive to a position, determined by software, which will be used as the datum, position (0,0).
Abort - Abort the motion the stage is undertaking.
About... - Information regarding the stage.
Close - Closes this dialog box.
