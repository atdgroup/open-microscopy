Microscope overview

The main microscope panel provides the most commonly used controls. More options are available via the 'Components' menu, as well as individual panels for the elements already shown. The 'Configure' menu offers some calibration possibilities.

camera settings
These controls are duplicated from the camera panel with the exception of 'No Binning'. 'Live' mode is a succession of integrated snaps at the exposure time indicated, so if an exposure time of several seconds is set then, after pressing 'Snap' to return to normal, the user will have to wait until the last integration has completed.

BF lamp
These controls are duplicated from the manual lamp panel.

stage
The stage section gives an updated display of the stage position. When the 'Store XYZ' button is pressed, the co-ordinates displayed will be stored to memory. At any subsequent time pressing 'GoTo XYZ' will drive the stage to these stored co-ordinates.

joystick
When the joystick is enabled the led to the right of the control will turn red.
