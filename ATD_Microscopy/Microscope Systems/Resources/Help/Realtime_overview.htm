Realtime overview



As the joystick moves (as long as the camera is in live mode) an image (low resolution) is displayed on the screen. One of the uses of this mode is to keep track of areas of the slide that have been visited and quickly return to them (by clicking on the low resolution image).

Load - loads an old stage scan (useful if the software suddenly crashes).
Reset - clears the overview image.
Disable/Enable - turns the overview image into a real-time overview panel.
