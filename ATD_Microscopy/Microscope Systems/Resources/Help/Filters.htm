Filters


These Filter set is composed of three different filters: Emission, Dichroic, and Excitation filter. The filter set is NOT physically assembled together like in the case of the fluorescent cubes.

The Filter are used only in the laser scanning modality (Microscope >> Mode >> Laser Scanning). In this modality the UV2 fluorescent cubes is chosen automatically by the software.



Configure - opens the dialog box (below) where different filter sets can be added, removed and edited.
Close - closes the dialog box.



Add - adds a new filter set.
Remove - remove a filter set from the list.
Edit - opens the dialog box (below) where specific information regarding the filter set is stored.
Recall - Loads a filter set list from file.
Save - Saves the current filter set list to a file
Close - closes the dialog box.




Filter set description- name of filter set.
Excitation Filter
Name - Name of Excitation filter
Excitation Min (nm)- Minimum wavelength allowed by the Excitation filter.
Excitation Max (nm) - Maximum wavelength allowed by the Excitation filter.
Dichroic Filter
Name - Name of Dichroic filter
Dichroic (nm) - Cut-off for the Dichroic filter.
Emission Filter
Name - Name of Excitation filter
Emission Min (nm)- Minimum wavelength allowed by the  Emission filter.
Emission Max (nm) - Maximum wavelength allowed by the  Emission filter.
OK- closes the dialog box.
